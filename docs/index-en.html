MessageHandler : a public system, Swiss-made and open source.

The MessageHandler is an utility that facilitates the message transit on the Sedex network. It implements functionalities that do not exist in the Sedex network, such as message compression and digitally signing documents.

It was originally a project of the Swiss confederation, but it has since then been taken by the state of Geneva. The MessageHandler is entierly developped in Java, and is nowadays largely used between departments and third-parties.

Open source licence : GPL v2

Since the project was made by a third party, the licence was kept as the editor set it. Therefore, the licence is the GPL v2. This licence has been written by the Free Software Foundation.

The State of Geneva is encouraging any IT community to contribute to the source code of this application through this platform.
